Module 4 · Lesson 13 – Community Documentation & Reddit Field Work

Observation mode, not promotion

The Daily Routine (10-15 Minutes Max)

Field operators do not promote. They observe, document, and intervene sparingly when failure modes are visible.

What you do daily:

  • Lurk comment threads where someone is already frustrated
  • Read for language, not topics
  • Exit threads that turn into advice dogpiles

What you do weekly:

  • Drop 0-2 links total (not per day, per week)
  • Never more than one per thread
  • Never in top-level posts

What you never do:

  • Never mention War Room by name
  • Never mention products or courses
  • Never DM first
  • Never follow up on your own comment

The Three Verbatim Responses

Pick one. Post once. Leave. No edits. No additions.

Response 1 — Default
"This might be off, but what you're describing has a name. I found this useful.
[link]"

Response 2 — If the thread is hostile
"Not advice—just a document that describes the failure mode you're pointing at.
[link]"

Response 3 — If stakes are explicit
"This doesn't fix it, but it does make the issue visible.
[link]"

What Counts as "Working"

You are not watching clicks or sales. You are watching for replies like:

  • "This is exactly what I'm dealing with"
  • "I didn't have words for this"
  • "This explains why it feels wrong"

If that happens:

  • You say nothing
  • If they ask "where do I go next?" you drop the resource link once
  • That's it

The Hard Stop

You are now in observation mode. Do NOT:

  • Improve the document
  • Add explanations
  • Create a website
  • Post summaries
  • Seek validation

If you feel the urge to "help it land," that urge is the signal your work describes.

Engagement Rules (From Field Calibration)

Never argue with:

  • Hostile or defensive users
  • Model superiority debates
  • Pricing tier or brand loyalty discussions

One reply per thread unless there is a clear invitation to continue.

Prefer reframing over rebuttal. Name the failure mode. Don't correct people pedantically.

If a reply would "burn cycles," disengage.

Interactive Exercise

Practice identifying failure modes and choosing the correct verbatim response:

Here is a Reddit thread excerpt: "I've been using ChatGPT for my consulting work and it's been great, but lately I've noticed it gives me different answers to the same question. Sometimes it recommends Strategy A, then two days later with the same prompt it recommends Strategy B. I'm starting to second-guess everything it tells me. Anyone else experiencing this?" For this thread: 1. Identify the failure mode being described 2. Determine: engage or do not engage (and why) 3. If engage: which of the 3 verbatim responses fits best 4. Draft the exact comment you would post (including which response template)

Checkpoint: Proof of Understanding

Find a real Reddit thread (or describe one you've seen) where someone is experiencing AI drift, conversational collapse, or unverified reliance. Provide: (1) Thread description or link, (2) Identified failure mode, (3) The EXACT verbatim comment you would post (using one of the 3 templates), or state "do not engage" with reason.
